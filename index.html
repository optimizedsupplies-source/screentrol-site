FAQ

Questions.

Screentrol uses Android's official ADB (Android Debug Bridge) — the same tool Android Studio uses. It does not modify your phone, collect any data, or require root access. The mirroring engine is scrcpy, a fully open source project by Genymobile with millions of users.
scrcpy is a command-line tool — powerful, but requires terminal knowledge to set up. Screentrol handles everything: bundling scrcpy, guiding you through developer options for your specific phone brand, managing connections, and providing a clean UI. You pay for the setup experience and wizard, not the underlying engine.
Screentrol v1.0.0 is Windows-only. Mac and Linux support are on the roadmap. In the meantime, scrcpy runs natively on those platforms if you're comfortable with the terminal.
